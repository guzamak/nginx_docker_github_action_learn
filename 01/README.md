### linux command
    sudo -i
      - เข้าสู่ root
    ls
      - เรียกดูไฟล์
      -l
        -สิทธิของ Folder
      -a
        -ไฟล์ที่ซ่อน
    mkdir
      - สร้างโฟลเดอร์
    cat
      - อ่านไฟล์
    cd
      - เข้า Folder
    apt get
      - โหลด package
    pwd
      - get ที่อยู่ปัจจุบัน
    sh
     - ใช้ shell (ลิงไม่ใช้ ลิงงง)
    bash
     - ใช้ bash (โหมด ปกติ)
    history
     - list คำสั่งก่อนหน้า
    nano
      - เเก้ไข้ข้อมูล
    touch	
     สร้างไฟล์ใหม่
    exit 
      - ออกจาก user หรือ root
    chmod
      - เปลี่ยน permissions ไฟล์
    chown
      - เปลี่ยน เจ้าของ ไฟล์
    whoami
      - เเสดง user ปัจจุบัน
    adduser 
      - สร้าง user ใหม่
    usermod
      - เเก้ไข user
